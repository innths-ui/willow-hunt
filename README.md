# AR Treasure Hunt App

A location-based augmented reality treasure hunt game where admins can place virtual objects for players to find and collect.

## Quick Start

Simply open `ar-treasure-hunt.html` in your browser or visit the live demo.

## Features

- **Admin Mode**: Place AR objects at real-world locations
- **Player Mode**: Find and collect objects to earn points
- **TV Stream Mode**: Display live gameplay on TV (1920x1080 @ 25fps)

## Usage

### For Players
1. Open the app on a smartphone
2. Switch to "Player Mode"
3. Allow location and camera permissions
4. Start hunting for treasures!

### For Admins
1. Stay in "Admin Mode"
2. Enter object type and location name
3. Click "Place Object" to add items at your current location

### For TV Display
1. Switch to "TV Stream" mode
2. Click "Start Broadcast"
3. View on TV in fullscreen (F11)
4. Watch live player activity at 25fps

## Requirements

- HTTPS connection (for GPS and camera access)
- Modern web browser (Chrome, Safari, Firefox)
- Smartphone for AR features
- Location permissions enabled

## Troubleshooting

**Location not working?**
- Ensure you're using HTTPS
- Check browser location permissions
- Try refreshing the page

**Camera not working?**
- Allow camera permissions when prompted
- Make sure you're on a secure (HTTPS) connection
- Use a supported browser
